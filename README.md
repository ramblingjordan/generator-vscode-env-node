# VSCode Node Development Environment Generator

## Steps
1. Get input
2. Make project folder `name`
3. `git init`
4. Create `README.md`
    - `name`
    - `description`
    - `node-version`
5. Create `Docker` file



- `git commit`
- Make github project
    - hub.github.com
- Push to github


## Input

- `name`
- `description`
- `node-version`
